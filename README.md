# Raju Erladinny Portfolio

Modern React + TypeScript portfolio built with Vite, Framer Motion and Lucide.

## Local setup

```bash
npm install
npm run dev
```

Run `npm run typecheck`, `npm run lint`, and `npm run build` before deployment.

Copy the resume to `public/Raju-Erladinny-Resume.pdf`. Edit content in `src/data.ts`.

## Deploy

Push to GitHub and import into Vercel or Netlify. Use `npm run build` and output directory `dist`. The included Vercel and Netlify SPA rules support direct links.
