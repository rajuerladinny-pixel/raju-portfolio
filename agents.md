# Project instructions

This is a React and TypeScript portfolio built with Vite.

## Inspection scope

Only inspect:

- src/
- public/
- package.json
- tsconfig.json
- tsconfig.app.json
- eslint.config.js
- index.html
- vercel.json
- README.md

Do not inspect:

- node_modules/
- dist/
- .git/
- package-lock.json
- \*.tsbuildinfo
- uploaded PDFs unless specifically requested

## Commands

Use:

- npm run typecheck
- npm run lint
- npm run build

Do not reinstall dependencies unless required.

## Content rules

- Use src/data.ts as the approved source for professional information.
- Do not invent experience, dates, clients, projects or achievements.
- Preserve the existing dark/light theme.
- Keep changes responsive and WCAG-accessible.
